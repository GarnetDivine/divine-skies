# Divine: Skies (Beta)

A RuneLite plugin by **O3 Studios** (Garnet Divine) that adds a configurable day/night sky cycle to Old School RuneScape. Works with the 117 HD plugin.

## Features

- **Day/Night Sky Cycle** — Smoothly transitions the skybox color between configurable day and night colors
- **Sunrise/Sunset Transitions** — Configurable transition duration (1-60 seconds) with an optional tint color for a warm sunset/sunrise glow
- **Sync to Local Clock** — Optionally map the cycle to your real local time, with configurable hour boundaries for sunrise, day, sunset, and night
- **Phase Tints** — Optional color tints for day and night phases, with adjustable strength
- **Brightness & Contrast Offsets** — Apply relative offsets to 117 HD's brightness and contrast per phase (day/night), smoothly interpolated during transitions
- **Underground Detection** — Automatically switches to a dark sky when entering dungeons, with instant transitions
- **On-Screen Overlay** — Shows current phase and time remaining, with a compact condensed mode option
- **Full Configurability** — Every color, duration, offset, and toggle is exposed in the config panel

## Requirements

- [RuneLite](https://runelite.net/)
- [117 HD Plugin](https://runelite.net/plugin-hub/show/117hd) (recommended, but the plugin also works without it)

## Configuration Sections

| Section | Description |
|---------|-------------|
| Cycle Timing | Day/night durations, transition time, local clock sync |
| Day Sky | Day sky color, brightness/contrast offsets, optional tint |
| Night Sky | Night sky color, brightness/contrast offsets, optional tint |
| Transition | Sunrise/sunset tint color and strength |
| Underground | Underground sky color and toggle |
| Overlay | Show/hide overlay, condensed mode |
| Advanced | Override 117 HD environment sky colors |

## Installation

Install from the RuneLite Plugin Hub — search for **"Divine: Skies"**.
